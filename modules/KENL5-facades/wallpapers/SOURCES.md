## Download Methodology

All wallpapers were downloaded using the following methods:

### Git Clone (Preferred)
```bash
# For GitHub repositories
git clone --depth 1 https://github.com/<org>/<repo>.git

# Example: CachyOS
git clone --depth 1 https://github.com/CachyOS/cachyos-wallpapers.git
```

### Direct Download (GitLab)
```bash
# For GitLab raw files
curl -L -o wallpaper.png \
  "https://gitlab.com/<org>/<repo>/-/raw/<branch>/path/to/wallpaper.png"
```

### Repository Browsing
For repositories without direct raw links:
1. Browse repository file tree
2. Identify PNG/JPG files
3. Download via `git clone` or GitHub raw URLs
