## Common Tasks (Visual Guide)

**Connecting to WiFi:**
```
1. Click network icon in system tray
2. See list of WiFi networks
3. Click your network
4. Enter password
5. Connected
```

**Installing an app:**
```
1. Open Software Manager
2. Search app name
3. Click Install
4. Enter password
5. Done
```

**Changing wallpaper:**
```
1. Right-click desktop
2. "Change Desktop Background"
3. Pick image
4. Click Apply
```

## Installation (Simplest in Linux)

1. Download ISO from linuxmint.com
2. Create bootable USB (use Rufus on Windows)
3. Boot from USB
4. Click "Install Linux Mint"
5. **Installer walks you through:**
   - Language
   - Keyboard layout
   - WiFi connection
   - "Install alongside Windows" or "Erase disk"
   - Username/password
   - Wait 15 minutes
   - Reboot
6. Done. Seriously.

**No command line, no confusion.**
